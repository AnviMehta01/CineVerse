# 🎬 CineVerse

CineVerse is a modern Android movie browsing application built using **Kotlin** and **Jetpack Compose**. The app allows users to explore trending movies, search through the collection, view detailed information about each movie, and manage a simple favorites list.

This project was developed as part of my Android Development learning journey and demonstrates UI design, navigation, state management, and Compose fundamentals.

---

## ✨ Features

- 🎬 Splash Screen
- 🏠 Home Screen with trending movies
- 🔍 Search movies by title
- 📖 Movie Details Screen
- ⭐ Movie ratings, genre, year and description
- ❤️ Add/Remove Favorites button
- 📱 Modern UI built with Jetpack Compose
- 🌙 Dark themed interface
